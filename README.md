# dotfiles

## ./setup
First installs [Brew](https://brew.sh/) and then the following:

1. [bash-completion](https://salsa.debian.org/debian/bash-completion)
1. [git](https://git-scm.com/)
1. [bash-git-prompt](https://github.com/magicmonty/bash-git-prompt)
1. [node](https://nodejs.org/en/)
1. [python 2.7.15](https://www.python.org/)
1. [postgresql](https://www.postgresql.org/)
1. [direnv](https://direnv.net/)
1. [vim](https://www.vim.org/)
1. [hub](https://hub.github.com/)
